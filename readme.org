** Init cascade

Init files (if existing) are loaded in this order:

- `~/.emacs.d/custom-user.el`
- `~/.emacs.d/setup/*.el`
- `~/.emacs.d/custom-user.el`
- `~/.emacs.d/init-user.el`
- `<project root>/.emacs/init.el`
- `<project root>/.emacs/init-user.el`
